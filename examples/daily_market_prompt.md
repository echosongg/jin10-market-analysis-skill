# 今日市场总览 Prompt

## 使用场景

当你想让 Hermes 做一次完整的今日多市场方向扫描时，直接发送以下 Prompt。

---

## Prompt

```
请使用 jin10-market-analysis skill，帮我分析今天的市场方向。

调用步骤：
1. list_calendar({}) 获取今日财经日历，标记 star>=3 的高星事件
2. list_flash({}) 获取最新快讯
3. 依次搜索以下关键词的快讯：美联储、通胀、非农、地缘政治、原油、黄金、日本央行、韩国、半导体、AI、韩元、欧佩克
4. 获取以下品种实时行情：XAUUSD、USOIL、USDJPY、EURUSD、USDCNH
5. 对今日 K 线趋势偏强的品种补充调用 get_kline（time=1h, count=24）
6. 判断当前市场 Regime（risk-on / risk-off / inflation_trade / rate_cut_trade / mixed）
7. 对 10 个市场方向评分（宏观25%+新闻催化25%+行情趋势20%+资金偏好15%+事件风险15%）
8. 输出中文分析报告，包含：今日结论、Regime 判断、评分总表、Top3 详情、重要待观察事件

请确保最后加上免责声明。
```

---

## 预期输出结构

```
一、今日结论
二、市场环境判断
三、多市场方向评分表（10个方向排名）
四、Top 3 重点方向详情（含利好/风险/观察条件）
五、其余方向简评
六、今日重要待观察事件
免责声明
```
