# 基金方向分析 Prompt

## 使用场景

当用户问"今天适合买哪类基金"、"哪个方向的基金值得关注"、"我想配置黄金/韩股/日股/港股方向基金"时使用。
本 Prompt 覆盖：黄金、原油、美股科技、港股互联网、A股成长、A股红利/防御、日股、韩股、债券方向。

---

## 韩股方向专项说明

韩股科技（三星电子/SK海力士）驱动因子优先级：

```
1. 全球 AI/HBM/DRAM 供需景气度（最强催化）
2. 美股科技板块整体走势（相关性高）
3. 韩元兑美元汇率（韩元贬值压制外资回报）
4. 韩国出口数据（半导体出口是先行指标）
5. 地缘风险（朝鲜、中美摩擦）
```

判断逻辑：
- AI/HBM 利好 + 韩元相对稳定 + 美股科技强 → 韩股方向可关注
- 韩元大幅贬值 + 外资流出 + 美股科技弱 → 韩股方向谨慎

---

## 日股方向专项说明

日股核心驱动因子：

```
1. 日元走势（日元贬值 → 日本出口企业受益 → 日股利好）
2. 日本央行政策（加息 → 日元升值 → 日股承压）
3. 全球风险偏好（risk-on 环境 → 日股受益）
4. 企业回购/治理改革进展
```

---

## 注意事项

- 所有评分和方向分析均为参考，不构成投资建议
- 若某方向数据不足，说明"数据有限，暂时中性观察"
- 不建议单方向重仓，不建议使用杠杆产品
- 高星级经济数据公布前，相关方向波动风险需重点提示
